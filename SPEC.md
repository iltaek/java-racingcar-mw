# 구현해야 할 사항 목록
 * Controller 구현
   - 레이싱카 게임을 실행시키고 얻어진 결과를 출력하는 역할을 수행하는 객체 클래스.
   - 레이싱카 목록과 게임 실행 횟수를 받아 게임을 실행시키고 전체 결과를 얻은 뒤 출력해주는 메소드.

# 구현 완료 한 사항 목록
 * Model 구현
   - 이름과 이동 거리를 가지는 차 객체 클래스.
   - 0 에서 9 사이의 랜덤 숫자를 생성할 수 있는 메소드를 가진 레이싱카 게임 객체 클래스.
   - 차 한 대에 대해서 0 에서 9 사이의 랜덤 값이 4 이상이면 전진하는 게임 메소드.
   - 차 여러 대에 대해서 게임을 한 판 하는 메소드. 
   - 레이싱카 게임 최종 우승자를 확인하고 관리하는 객체 클래스.
   - 레이싱카 목록을 받아 최종 우승자 목록을 반환해주는 메소드.
 * View 구현
   - 레이싱카 목록을 받아 중간 진행 상태를 그림으로 표현해주는 메소드.
   - 레이싱카 최종 우승자 목록을 받아 최종 진행 결과를 알려주는 메소드.
 * View 개선
   - View 객체에서 레이싱카 게임 최종 우승자를 직접 계산하는 내용 제거.
